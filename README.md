# kalp-atisi-demo
Map-based message scheduling and referral system demo project.  Harita tabanlı mesaj planlama ve referans sistemi demo projesi.

---------------------------------------------------------------------------------------------------------------------------------------------------
ENGLISH 

Kalp Atışı Demo is a full-stack web application prototype that showcases how a modern, user-focused platform can combine map-based interaction, scheduled message delivery, user accounts, and a referral-driven growth system into a single coherent experience.

At its core, the project is designed around a simple but powerful idea: users can create meaningful “heartbeat” messages and schedule them to be delivered at the right time, while the platform visualizes activity through an interactive map and real-time statistics. This creates a product feel that is both emotional and data-driven — a rare combination for a demo-level application.

Key Highlights

Authentication & Account System

Supports user onboarding with a standard registration flow.

Includes Google OAuth login, demonstrating real-world identity provider integration and secure session handling.

Interactive Map Experience

Uses a map interface to visually represent user activity with pins.

Clicking markers reveals structured detail views (e.g., sender identity + message context), designed to feel like a live platform dashboard rather than a static page.

Scheduled Message Workflow

Users can write a message (“Kalp Atışı”), provide delivery details, and schedule it.

The flow is intentionally built like a production product: clean form UX, validation mindset, and a clear progression from creation → scheduling → management.

Referral System + Growth Mechanics

Includes referral code logic and usage tracking to represent platform growth strategies.

Designed with measurable performance indicators (counts, earnings logic, usage metrics), making it closer to a startup-ready concept than a basic school project.

Profile & User Panels

Profile section consolidates user information, referral codes, certificates, and earning-related UI components.

The UI structure is modular and expandable — ready for future improvements like payments, notifications, and admin tooling.

Why This Project Stands Out

This repository is not just a set of pages — it is a product-oriented prototype. The structure reflects real-world application thinking: clear flows, consistent pages, functional modules, and an experience built for actual users. Even without a payment gateway or deployment, the project demonstrates a mature approach to system design, feature planning, and user experience.

TÜRKÇE 

Kalp Atışı Demo, modern bir web platformunun nasıl kurgulanabileceğini gösteren, uçtan uca düşünülmüş bir full-stack prototip uygulamasıdır. Proje; harita tabanlı etkileşim, zamanlanmış mesaj sistemi, kullanıcı hesapları ve referans ile büyüme mekanizmasını tek bir ürün deneyiminde birleştirir.

Projenin temel fikri hem sade hem güçlü: kullanıcılar “kalp atışı” niteliğinde anlamlı bir mesaj oluşturur, ileri bir tarihe planlar ve platform bu etkileşimi harita üzerinde görselleştirerek canlı bir sistem hissi oluşturur. Bu yaklaşım, duygusal bir ürün fikrini ölçülebilir istatistiklerle bir araya getirir — demo projelerinde nadir görülen bir “ürün mantığı” sunar.

Öne Çıkan Özellikler

Kimlik Doğrulama & Hesap Sistemi

Kayıt olma akışıyla kullanıcı onboarding’i destekler.

Google OAuth ile giriş entegrasyonu bulunur; gerçek uygulamalarda kullanılan güvenli oturum yönetimi mantığıyla kurgulanmıştır.

Etkileşimli Harita Deneyimi

Harita üzerinde pin’ler ile platform aktivitelerini görselleştirir.

Pin’e tıklanınca detayların düzenli biçimde gösterilmesi, projeyi “sayfa sayfa gezen site” olmaktan çıkarıp “canlı platform paneli” hissine yaklaştırır.

Zamanlanmış Mesaj Akışı

Kullanıcı mesajını (“Kalp Atışı”) yazar, gerekli bilgileri girer ve gönderim tarihini planlar.

Akış, üretim ortamı düşüncesiyle hazırlanmış gibidir: temiz form yapısı, ilerleyen adımların netliği ve yönetilebilirlik mantığı.

Referans Sistemi & Büyüme Mekanizması

Referans kodu üretimi ve kullanım mantığı içerir; gerçek dünyadaki growth stratejilerini temsil eder.

Sayaçlar, kazanç mantığı, kullanım metrikleri gibi parçalar projeyi basit bir öğrenci demosundan çıkarıp “startup fikri prototipi” seviyesine taşır.

Profil & Kullanıcı Panelleri

Kullanıcı bilgileri, referans kodu, sertifikalar ve kazanç bileşenleri tek panelde toplanır.

Arayüz yapısı modüler ve genişletilebilir tasarlanmıştır; ödeme, bildirim, admin geliştirmeleri gibi özelliklere hazırdır.

Bu Projeyi “Profesyonel” Yapan Şey

Bu repo yalnızca birkaç sayfa ve formdan oluşan bir çalışma değil; ürün odaklı bir prototiptir. Sayfa akışları, modüler yapı, özelliklerin birbirine bağlanma şekli ve kullanıcı deneyimi yaklaşımı, gerçek bir platformun iskeletini taşır. Hosting veya ödeme entegrasyonu olmadan bile proje; sistem tasarımı, özellik planlama ve UX düşüncesi açısından profesyonel bir perspektifi açıkça yansıtır.
